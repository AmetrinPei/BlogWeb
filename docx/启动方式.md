这是前后端分离项目，需要 **MySQL + 后端 + 前端** 都跑起来，浏览器访问前端地址。

## 1. 前置条件

- **MySQL 8.x**（本地 `127.0.0.1:3306`）
- **Java 21**
- **Node.js**（前端用）

数据库默认配置（`backend/src/main/resources/application.yml`）：

- 库名：`blog`（一般会自动创建）
- 用户：`root`
- 密码：`root`（可用环境变量 `DB_PASSWORD` 覆盖）

## 2. 启动后端

开一个终端：

```powershell
cd "d:\workspace\task\mini program\backend"
.\mvnw.cmd spring-boot:run
```

成功后后端在：**http://localhost:8080**

## 3. 启动前端

再开一个终端：

```powershell
cd "d:\workspace\task\mini program\frontend"
npm install
npm run dev
```

成功后前端在：**http://localhost:5173**  
开发时 `/api` 会代理到后端 `8080`。

## 4. 怎么访问网页

| 页面 | 地址 |
| --- | --- |
| 博客首页 | http://localhost:5173/ |
| 文章列表 | http://localhost:5173/articles |
| 关于我 | http://localhost:5173/about |
| 后台登录 | http://localhost:5173/admin/login |
| 后台（登录后） | http://localhost:5173/admin |

默认管理员账号：

- 用户名：`admin`
- 密码：`admin123`

## 注意

- 只开前端、不开后端时，页面能打开，但文章等接口会失败。
- 在浏览器里访问的是 **5173**，不是 8080。
