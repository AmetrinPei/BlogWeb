Spring Boot 3 快速入门：从零搭 REST

发布于 2026-07-10 · 小橘

Spring Boot 3 基于 Java 17，把配置和部署变得更轻松。今天用治愈系心情，搭一个最小可用的博客 API。

统一响应

封装 code / message / data，让前端永远知道怎么读结果。

{"code":0,"message":"ok","data":{...}}

公开读接口无需登录；管理写接口带上 JWT，像给后台加了一把薄荷绿的小门锁。
